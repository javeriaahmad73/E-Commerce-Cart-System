# OOP E-Commerce Cart System

## Project Description
This project is a simple E-Commerce Cart System implemented in Python using Object-Oriented Programming (OOP) principles.  
It allows users to add products to a shopping cart, remove products, calculate subtotal, and apply different pricing strategies such as percentage discounts.

The system is designed with clean code practices and separation of concerns to keep business logic independent from the command-line interface.

---

## Features
- Add products to shopping cart
- Remove products from cart
- Calculate subtotal of items
- Apply pricing strategies
- Percentage discount support
- Unit testing for core logic
